Case definition on the same line as RBRACE (REPY-0004)
==========================================================

Description
-----------
This error occurs when a ``case`` definition is on the same line as a closing RBRACE.

Incorrect:
----------
.. code-block:: repy

    match(num) {
        case 1{
            pass
        } case _{      <---- REPY-0005
            pass
        }
    }

Correct:
--------
.. code-block:: repy

    match(num) {
        case 1{
            pass
        }          <----
        case _{
            pass
        }
    }
